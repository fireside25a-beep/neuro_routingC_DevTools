# neuro_routingC_DevTools
NeuroRoute DevCLI is a portable C11 tool built around a deterministic routing library. It detects project types, creates starter projects, selects installed toolchains, and safely builds, runs, tests, cleans, and edits all C, C++, Python, Rust, JavaScript, Go, Zig, and Java projects on macOS and Linux, with clear ambiguity and missing-tool reports.
